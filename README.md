# 🚀 Gracias Jeremy CLI

**Automatizador inteligente para Git y GitHub** — by Jeremy

---

## ✨ Características Pro
- 🧹 **Limpieza Total:** Pantalla limpia cada vez que inicias.
- 🎨 **Interfaz en Colores:** Morado (inicio), Verde (éxito), Amarillo (preguntas), Rojo (error).
- 📦 **Git Init & Commit:** Inicializa y crea el primer commit obligando a nombrar la tarea.
- 🛠️ **Auto-Instalación:** Si no tienes `gh` (GitHub CLI), el script lo instala por ti.
- 🔑 **Login Express:** Abre el navegador automáticamente para asociar tu cuenta.
- 🌐 **Creación de Repos:** Crea el repo (público/privado) en GitHub desde la terminal.

---

## 📦 Instalación
Para tenerlo como comando global en cualquier proyecto:

```bash
npm install -g gracias-jeremy
```
